# Qlub Social

![Qlub Social Logo](https://qlub.social/system/site_uploads/files/000/000/006/@2x/830f401aeeb0e080.png)

Qlub est la plateforme sociale québécoise qui mise sur l’engagement local et les échanges authentiques.

- [Qlub Social](https://qlub.social)
- [Contactez-nous](mailto:bonjour@qlub.social)

## La Langue Française du Québec

Le Ministère de la Programmation et du Code Source Ouvert du Québec (MPCSOQ) ne nous a pas alloué assez de budget pour créer notre propre langage de programmation Québécois. Le budget a certainement été alloué à la SAAQ... Nous avons donc dû utiliser le langage de programmation qui nous permet de le modifier pour qu’il soit en accord avec les normes et lois de la langue française du Québec. Nous avons déterminé que le langage de programmation Cé (aussi connu sous le nom « C ») serait suffisant.

## Système Minimum

Vous avez besoin au minimum d’un système qui serait capable de rouler Fenêtre XP (aussi connu sous le nom « Windows XP »). Par contre, vous aurez besoin d’installer une distribution de Linuxe (aussi connu sous le nom « Linux ») Québécoise car ProgrammeMicro (aussi connu sous le nom « MicroSoft ») est une compagnie américaine et le créateur de Linuxe Léon Dorval (aussi connu sous le nom « Linus Torvalds ») ne l’est pas. Le Ministère de la Programmation et du Code Source Ouvert du Québec (MPCSOQ) nous offrira sûrement assez de budget pour construire notre propre système québécois...?

## Construire L'Application

Vous devez utilisé l'outil faire (aussi connu sous le nom « make ») pour construire l'application écrit en Cé. Voici la démarche que vous devez suivre:

```
faire -f FaireFichier
```

## Exécuter L'Application

C'est simple, vous devez simplement exécuter l'exécutable produit dans l'étape de construction:

```
./qlub
```
